# Checkout / Offers Page — Design Concepts

Four self-contained checkout/offers page concepts (same offer & functionality, different visual direction) for a client to choose from. Open `index.html` for the gallery.

- **option-1-midnight** — dark premium / high-end tech
- **option-2-luxe** — editorial luxury / clean premium
- **option-3-pop** — vibrant DTC / energetic
- **option-4-clinical** — clinical trust / conversion-optimized

Each is a single static `index.html` (inline CSS+JS, no build) with selectable supply tiers, progressively unlocking free gifts, and a live order summary. Deploy the folder to any static host.
